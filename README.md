## Repository Structure

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/komalapatitejaswini/python-DE.git
   ```

2. Navigate to the project folder:

   ```bash
   cd python-DE
   ```

3. Run any Python file:

   ```bash
   python filename.py
   ```
